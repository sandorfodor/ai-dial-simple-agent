# DIAL AI Simple Agent Task
Python implementation for building AI-powered chat applications using the DIAL API with advanced tool integration.

Create .env based on the .env.example in the task folder.

Update `run.py` based on which task you want to execute.

### Build & Run

```bash
docker build -t python-docker .;docker run -it --rm --network ai-dial-simple-agent_my-network python-docker
```
